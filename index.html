HI THERE

I do A LOT of stuff with AWS. I do A LOT of stuff with InterSystems technology. This spawned from the need to upload marshaled HL7 data to the Machine Learning Models in S3, among other things.

So this is a the way to interact with AWS from HealthShare, Cache, or InterSystems solutions, whatever. In a nutshell, it's a development pattern around the JAVA SDK and wrapped up via the Java Gateway. I slapped it into a code package and surfaced it through a HealthShare production.

Basically, if you understand this...
